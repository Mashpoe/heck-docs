Operator Overloading

Operator overloading extends the functionality for objects of a given class type.

Many operators can be overloaded, such as addition:

class Foo {

    value: int
    
    func Foo(val: int) {
        value = val
    }

    // overload addition for two "Foo" objects
    func operator+(other: Foo) {
        return Foo(value + other.value)
    }
}

let foo = Foo(3) + Foo(2)

print(foo.value) // prints 5

In the example above, the addition (+) operator is overloaded to allow addition between two "Foo" objects. This creates a new "Foo" object whose value is the sum of the previous two objects' values.
