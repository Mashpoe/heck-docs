If Statements

If statements allow conditional branching in your code, and work identically to if statements in most other programming languages. If the result of an if statement's condition expression is true, the corresponding block of code is executed:

if condition /* (expression) */ {
    // code
}

If statements can also include an else block, which executes if the result of the condition is false:

if condition /* (expression) */ {
    // code
} else {
    // other code
}

Additionally, if statements can include an arbitrary number of else-if blocks. If the result of the preceding condition is false, then the condition of the else-if block will be checked and the corresponding block of code will be executed accordingly:

if condition1 /* (expression) */ {
    // code
} else if condition2 /* (expression) */ {
    // code
} else if condition3 /* (expression) */ {
    // code
}

This is known as an "if-else ladder."

If-else ladders can also end with an else block:

if condition1 /* (expression) */ {
    // code
} else if condition2 /* (expression) */ {
    // code
} else if condition3 /* (expression) */ {
    // code
} else {
    // code
}
