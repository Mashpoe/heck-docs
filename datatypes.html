Data Types

Different data types can do different things. Some primitive data types include:

// integer
let x: int = 4

// string
let y: string = "Hello, World!"

// float
let z: float = 0.3

Heck is statically-typed, so data types cannot be changed:

let x = ""

x = 2

Heck uses type inference, which allows the language to deduce data types in certain situations.

Data Types:

Integers store a positive or negative whole number.

let x: int = 3

x = x + 1

Floats can store positive and negative decimal values.

let x: float = 1.0

x = x + 3.2

Bools can store true or false data.

let x: bool = true

x = false

Strings can store bits of text. You cannot do math with number strings.

let x: string = ""

x = "Hello, World!"

Class types are used to create objects, which are instances of classes

Callbacks can store a function to a variable

func add(a: int, b: int) {
    return a + b
}

let x = add

print(x(4, 6)) //prints 10
