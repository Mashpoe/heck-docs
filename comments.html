Comments

Comments are bits of text that don't do anything in the code but help with other things. Here is how you make a line comment:

// these can only take up one line

Block comments can take up multiple lines, but can also go in the middle of one.

/* niceboii
sup
ok
byeeeeeeee */

Unlike most other languages, heck allows you to nest block comments.

/* look! /* What? */ */

/* cool
/* nice
*/
*/
