Type Casting

A type cast converts a value from one data type to another.

Not all types can be converted to one another; for example, a string cannot be directly converted to a float.

Some type conversions can be done implicitly, such as integer to boolean, but there are many conversions that must be done explicitly.

Type conversions can be done explicitly using the as operator, which takes an expression as its left operand and a data type as its right operand:

let x: float = 6.0

let y: int = x as int
